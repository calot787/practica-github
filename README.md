##Documentacion
GitHub es una plataforma en la nube para desarrolladores que aloja proyectos de software usando Git,

Los repositorios en GitHub se clasifican principalmente por su visibilidad en públicos (accesibles para cualquiera en internet, ideales para código abierto) y privados (restringidos a colaboradores específicos, ideales para proyectos confidenciales o empresariales).
